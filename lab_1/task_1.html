Задание: Напишите функцию, которая будет проверять, является ли переданное ей число квадратным, то есть являющимся квадратом некоторого целого числа. Функция должна возвращать true или false. Пример:

isSquare(-1) // => false
isSquare( 3) // => false
isSquare( 4) // => true
isSquare(25) // => true
isSquare(26) // => false
